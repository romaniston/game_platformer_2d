# game_platformer_2d
A platformer game based on DOOM on PyGame
________________________________
Управление:
W, A, D - Ходьба и прыжок.
LSHIFT и RSHIFT - Бег.
Пробел - Стрельба.
Цифры - Выбор оружия.
